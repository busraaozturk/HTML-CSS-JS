# 🎨 Renkli Butonlar Galerisi – CSS Hover, Transition & Animasyonlar
Bu proje, modern web arayüzlerinde kullanılan **şık buton tasarımlarını** sergileyen interaktif bir galeri sunar.  
CSS, JavaScript ve temel DOM manipülasyonları ile hazırlanmış bu çalışma, hem tasarım hem de etkileşim açısından eğlenceli bir deneyim sunar.

## 🚀 Proje Özellikleri
- Renkli ve stilize edilmiş butonlar  
- Üzerine gelince yükselme (hover) efekti  
- Basılınca küçülme efekti  
- Ripple (dalga) animasyonu  
- Rastgele tıklama animasyonları (`bounce`, `shake`, `flip`)  
- 3D hover efekti  
- Koyu/Açık Tema değiştirici  
- Temaya göre ikon (svg) değiştirme (Güneş 🌞 / Ay 🌙)

## 📂 Proje Dosyaları
- `button.html` → Anasayfa HTML yapısı
- `button.css` → Buton stil ve animasyon dosyası
- `button.js` → Ripple efekti, tema değiştirici ve tıklama animasyonlarını sağlayan JavaScript dosyası

## 💡 Kullanılan Teknolojiler
- HTML5
- CSS3 (Flexbox, Animasyonlar, Transitions)
- JavaScript (DOM Events, Tema Değişimi, Animasyon Trigger)
