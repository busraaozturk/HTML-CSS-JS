# Kişisel Profil Kartı

Bu proje basit bir **kişisel profil kartı** tasarımını içerir.  
HTML ve CSS kullanılarak responsive (mobil uyumlu) şekilde geliştirilmiştir.

## 🔥 Proje Özellikleri
- Modern ve sade tasarım
- Responsive uyum (mobil, tablet, desktop)
- Sosyal medya ikonları (GitHub, Instagram, YouTube, LinkedIn)
- Hover efektleri ve animasyonlar
- Açıklama (bio) alanı
- Temiz ve düzenli kod yapısı

## 📁 Kullanılan Teknolojiler
- HTML5
- CSS3
- Responsive Web Design (Media Queries)
- SVG ikonlar

## 📁 Dosya Yapısı
- **profilkart.html** → HTML yapısı
- **profilkart.css** → Kartı stillendiren CSS
