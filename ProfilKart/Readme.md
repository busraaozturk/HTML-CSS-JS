# 👩‍💼 Çalışan Profil Kartı

Bu proje, kullanıcıya ait bir **çalışan profil kartı** tasarımını içermektedir. HTML ve CSS kullanılarak oluşturulmuş bu bileşen; kullanıcı avatarı, pozisyon bilgisi, ikonlar ve modern bir kart yapısı ile estetik bir görünüme sahiptir.

## 💡 Özellikler
- Tamamen **HTML** ve **CSS** ile oluşturulmuştur.
- Responsive kart yapısı
- Kullanıcı ünvanı ve puan bilgisi bulunmaktadır.
- Svg ikonları ile görselleştirme

## 🧩 Kullanılan Teknolojiler
- HTML5
- CSS3 
- SVG ikonlar
- Responsive yapı (mobil uyumlu değil ama geliştirilebilir)

## 📁 Dosya Yapısı
- **profilkart.html** → HTML yapısı
- **profilkart.css** → Kartı stillendiren CSS
