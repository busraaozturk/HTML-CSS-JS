# Scroll Snap Kart Slider Projesi

Bu proje, HTML ve CSS kullanılarak yatay olarak kaydırılabilen, kart tasarımlarının **scroll-snap** özelliğiyle ekrana düzgün bir şekilde hizalandığı **minimalist** bir kullanıcı arayüzüdür.

## 💡 Özellikler
- Tamamen **HTML** ve **CSS** ile oluşturulmuştur.
- Yatay scroll ile kaydırılabilir kart yapısı
- `scroll-snap` özelliği ile her kart ekranın ortasında hizalanır
- Renkli ve stilize kartlar
- Dikeyde ve yatayda ortalanmış içerik
- Özel scroll bar tasarımı
- Responsive'e uygun yapı için kolay genişletilebilirlik

## 🧩 Kullanılan Teknolojiler
- HTML5
- CSS3 (Flexbox, Scroll Snap)
- SebKit custom scrollbar stilleri

## 📁 Dosya Yapısı
- **scrollcard.html** → HTML yapısı
- **scrollcard.css** → Kartları ve container'ı stillendiren CSS
