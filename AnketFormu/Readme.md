# 📝 Anket Formu Projesi

Bu proje, kullanıcıların temel kişisel bilgilerini toplayan şık ve kullanıcı dostu bir **Anket Formu** içerir. Form HTML, CSS ve JavaScript kullanılarak geliştirilmiştir.

## 🚀 Proje Özellikleri
- **Zorunlu Alanlar** kontrolü
- **Doğrulama** (Telefon numarası formatı, "Diğer" hobi bilgileri girildiğinde ekstra input alanı)
- **Karakter Sayacı** (Textarea alanı için)
- **Yalnızca Sayı Girişi** (Telefon numarası için)
- **Yükleniyor Animasoyunu** (Form gönderiminde)
- **Responsive Tasarım** (Mobil uyumlu)

## 📂 Proje Dosyaları
- `anketform.html` → Formun HTML yapısı
- `anketform.css` → Formun stil dosyası
- `anketform.js` → Formun dinamik kontrollerini sağlayan JavaScript dosyası

## 💡 Kullanılan Teknolojiler
- HTML5
- CSS3 (Flexbox, Responsive tasarım, Animasyon)
- Vanilla JavaScript (Saf JS)

## 📌 Notlar
- Telefon numarası alanı sadece rakam kabul eder ve en fazla 11 hane olabilir.
- "Diğer" hobisi seçilirse ek bir metin kutusu görünür ve doldurulması zorunludur.
- Form verileri şu anda herhangi bir sunucuya gönderilmemektedir (demo amaçlıdır).
