# Responsive Kart Tasarımı

Bu proje, HTML ve CSS kullanılarak oluşturulmuş **modern ve duyarlı (responsive)** bir ürün kartı tasarımı sunar. Kart üzerinde hover efekti, ikonlar, etiketler ve mobil uyumlu stil detayları bulunmaktadır.

## 💡 Özellikler
- Tamamen **HTML** ve **CSS** ile oluşturulmuştur.
- **Responsive** yapı sayesinde her ekran boyutunda merkezlenmiş görünüm sağlar.
- Hover'da arka plan görseline zoom efekti bulunmaktadır.
- **Modern** ve **şık** görsel yapısı oluşturulmuştur.
- SVG ikonlar ile vektörel ve kaliteli grafik kullanımı

## 🧩 Kullanılan Teknolojiler
- HTML5
- CSS3
- SVG (ikonlar için)

## 📁 Dosya Yapısı
- **card.html** → HTML yapısı
- **card.css** → Animasyon ve görünüm için CSS kodları
